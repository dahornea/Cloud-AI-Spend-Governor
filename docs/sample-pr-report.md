# Sample GitHub PR Report

This is a realistic sample of the report shape generated by the MVP. Exact values depend on the scan input, pricing configuration, and policy thresholds.

```md
<!-- cloud-ai-spend-governor-report -->

## Cloud & AI Spend Governor Report

Status: FAIL

| Metric | Value |
|---|---:|
| Estimated monthly impact | +EUR 342.00 |
| Baseline monthly cost | EUR 0.00 |
| Proposed monthly cost | EUR 342.00 |
| Environment | dev |
| Confidence | Medium |
| Final decision | FAIL |

### Main cost drivers

| Resource | Type | Change | Estimated monthly cost |
|---|---|---:|---:|
| redis-prod | azurerm_redis_cache | Added | +EUR 310.00 |
| api-plan | azurerm_service_plan | Added | +EUR 30.00 |
| storage-dev | azurerm_storage_account | Added | +EUR 2.00 |

### Policy findings

| Rule | Result | Message |
|---|---|---|
| dev-delta-limit | Block | Monthly delta is EUR 342.00, which is above the dev threshold of EUR 100.00. |

### Assumptions

- MonthlyHours = 730
- Currency = EUR
- PricingSource = LocalStaticCatalog or AzureRetailPricesApi, depending on configuration
- DefaultRegion = westeurope
- ArmTemplateDiffMode = DesiredStateOnly when compiled ARM JSON is used

### Pricing metadata

| Resource | Source | Catalog version | Match type | Fallback reason |
|---|---|---|---|---|
| redis-prod | Local Pricing Catalog | 2026.07.01 | ExactRegionSkuMatch | |
| api-plan | Local Pricing Catalog | 2026.07.01 | ExactRegionSkuMatch | |
| storage-dev | Local Pricing Catalog | 2026.07.01 | ResourceTypeFallback | usage assumption applied |

### Analysis source

Bicep compiled ARM JSON was detected and used for this report.

### Recommendation

- Use a cheaper Redis SKU in dev or configure environment-specific SKUs before merging.
- Review App Service plan sizing for non-production environments.

Dashboard report: http://localhost:5102/?analysisId=<scan-id>
```

In local simulated mode, the report is stored as simulated GitHub publishing metadata. In real GitHub App mode, the app updates or creates a PR comment and can update a check run when enabled.
